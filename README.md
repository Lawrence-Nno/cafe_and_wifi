# Cafe and Wifi
This website shows the list of all the cafes and work friendly spaces in the city and the features.
The city used in this project is London but you can easily add any cafe of any city you want.

## Tech
- Python
- Flask
- Jinja
- HTML
- CSS
- Javascipt
- MySQL
